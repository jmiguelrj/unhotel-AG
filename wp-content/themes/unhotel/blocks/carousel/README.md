# Carousel Block

Flexible Gutenberg block for displaying carousels of images, testimonials, or blog posts.

## Features
- Source: images, testimonials, or posts
- Layout: full width or container
- Options: items per page, arrows, dots
- Dynamic PHP rendering
- Flickity integration

## Files
- `block.js`: Block registration and editor UI
- `carousel.scss`: Block styles (SCSS)
- `carousel.css`: Compiled CSS
- `index.php`: PHP render callback
- `block.json`: Block metadata
